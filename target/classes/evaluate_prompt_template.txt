# 评分标准
- **技术正确性 (techAccuracy)**: 答案中技术概念、原理、实现方法的准确性
- **知识深度 (techDepth)**: 对技术理解的深入程度，是否触及底层原理和本质
- **逻辑严谨性 (logicCohesion)**: 回答的逻辑结构是否清晰，论证是否严密
- **岗位适配度 (jobMatchScore)**: 答案体现的能力与目标岗位的匹配程度

# 评分要求
1. 保持客观、公正、严谨的评分态度
2. 基于知识库内容作为参考答案进行对比
3. 每个指标的分数范围为 0-10 分，可以有小数（如 7.5）
4. 避免分数虚高，要真实反映候选人的水平

# 输出格式要求
**必须且只能返回纯 JSON 格式数据**，不要包含任何 Markdown 标记（如json），不要有任何额外说明文字
